# Permalinks{#permalinks}

Use seu encurtador de URL bit.ly para personalizar o conteúdo permalinks.

Os links diretos permitem se vincular a comentários arquivados ou conteúdo em seus fluxos. Para habilitar esse recurso, o Livefyre usa um formato de URL padrão:** fyre.it**. Esse formato pode ser personalizado para corresponder ao seu formato bit.ly padrão.

## Exibição de pré-link {#section_mxd_wwg_wy}

Se os usuários clicarem em um permalink de uma rede social para um aplicativo de conteúdo do Livefyre, eles serão direcionados para o fluxo de conteúdo do Livefyre. O conteúdo será exibido em uma exibição de detalhamento do fluxo, logo abaixo do editor, com o contexto completo exibido. Uma linha azul será mostrada abaixo do avatar para o conteúdo permalinked. (Essa linha azul desaparecerá após alguns segundos.)

(Se o conteúdo com permissão for excluído, uma mensagem para esse efeito será exibida na exibição de detalhamento.)

## Personalizar pré-links {#section_uyn_5wg_wy}

Se você quiser criar permalinks personalizados para comentários ao compartilhar conversas, poderá usar sua conta Bitly, em vez do URL padrão encurtado do Livefyre (fyre.it). Usar sua conta Bitly ajudará a lembrar os usuários de que eles visitarão o conteúdo do seu site ao clicar em um link permanente.

Para implementar esse recurso, vá para **[!UICONTROL Studio > Settings > Integration Settings > Social Sharing]** e insira sua Chave de API Bitly e nome de usuário.

>[!NOTE]
>
>Para evitar URLs desnecessários criadas em seu nome, os chamadas de Bitly são gerados conforme necessário.
