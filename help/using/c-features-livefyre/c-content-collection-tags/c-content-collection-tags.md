# Tags de coleção e conteúdo{#collection-and-content-tags}

Adicione tags de palavra-chave a coleções e conteúdo, para categorizar e organizar o conteúdo para fácil recuperação.

A atribuição de tags permite filtrar pesquisas por uma tag específica e extrair dados para a API Heat Index.

As tags podem ser adicionadas pelos painéis **[!UICONTROL Content]** e **[!UICONTROL Collections]** no Studio ou diretamente aos Metadados da coleção (usando o objeto collectionMeta) ao inicializar ou atualizar a coleção com o Livefyre.

## Uso de tags de coleção com a API Heat Index {#section_uhd_frk_wy}

A API Heat Index aceita um parâmetro Tag, que permite recuperar a integridade relativa de qualquer Coleção com essa tag em um site ou em toda a rede.

Para obter detalhes sobre a implementação, consulte a documentação técnica da API Heat Index.
