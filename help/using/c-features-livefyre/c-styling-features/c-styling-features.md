# Recursos de estilo{#styling-features}

Você pode criar um estilo para seus aplicativos Livefyre e se comportar de várias maneiras ou alterá-los para que pareçam com o resto da campanha da sua marca.

## Estilo padrão {#section_yjl_rfs_d1b}

Você pode criar um estilo para um aplicativo no Studio no App Designer sem precisar usar o Livefyre.js para personalizar o aplicativo.

* [Comentários](/help/using/c-about-apps/c-comments/c-comments.md)
* [Placa de recurso](/help/using/c-about-apps/c-feature-card-app/c-feature-card-app.md#c_feature_card_app)
