# Método do Site getUrn{#geturn-site-method}

Este método retorna o URN para este site.

## Exemplo de Java {#section_nyl_ycs_rz}

Uso:

```
site.getUrn();
```

Exemplo de saída:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
