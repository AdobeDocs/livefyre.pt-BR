# Método do site geturn{#geturn-site-method}

Esse método retorna o URN desse site.

## Exemplo Java {#section_nyl_ycs_rz}

Uso:

```
site.getUrn();
```

Saída de amostra:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
